# SyncMySubs

A simple tool to shift subtitle timings in a SRT file.

How to use 

- Open [the website](https://thanzeelhassan.github.io/SyncMySubs) in a browser.
- Choose an `.srt` file, enter seconds to shift (positive to delay, negative to advance), then click `Apply & Download`.
